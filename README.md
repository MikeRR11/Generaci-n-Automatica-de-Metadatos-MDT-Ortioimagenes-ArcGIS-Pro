# Generaci-n-Automatica-de-Metadatos-MDT-Ortioimagenes-ArcGIS-Pro
Aplicativo en ArcGIS Pro para la generación automática de metadatos de MDT y ortoimágenes, conforme a ISO 19139 y los estándares de la ICDE. Automatiza la documentación geoespacial, garantizando interoperabilidad, calidad y cumplimiento normativo en un archivo de metadatos XML según ISO 19139.
